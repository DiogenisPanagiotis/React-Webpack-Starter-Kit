# Getting started with React and Webpack

This is the bare minimum requirements to get started with React, Webpack and Babel (includes webpack-dev-server which allows for hot reloading). 

## Dependencies
- react
- react-dom
- babel-core
- babel-loader
- babel-preset-es2015
- babel-preset-react
- webpack
- webpack-dev-server

## Instructions
1. npm install
2. npm start
